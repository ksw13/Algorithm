## 코드
```java
class Solution {
    public int solution(int n) {
        int answer = 0;
        
        for(int i=1; i<=n; i++){
            int temp=0;
            for(int j=i; j<=n; j++){
                temp+=j;
                if(temp>n) break;
                if(temp==n){
                    answer+=1;
                    break;
                }
            }
        }
        
        return answer;
    }
}
```
